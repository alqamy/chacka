# Chacka
 Content Management System using **nodejs** and **mongodb**

![693](https://cloud.githubusercontent.com/assets/6148982/3945885/e8b05c86-2654-11e4-84dc-da9c2c7da6ac.jpg)



## Pre Requirement
* NodeJs
* MongoDB


## Installation
```
  npm install -g chacka
```


## Running

```
chacka folder-name
```
